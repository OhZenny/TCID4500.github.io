    • Creating and Cloning a GitHub Repository

    Setting up a GitHub repository (repo) is like creating a shared workspace for teamwork, keeping everyone on the same page. When you clone it to your computer, it's like bringing a copy of that workspace to your desk. This makes it easy to work together, try out new ideas, and contribute to the project. GitHub helps you organize your work and collaborate with others smoothly.

    Prerequisites...

    1. Have a GitHub account.
    2. Have Git installed and configured.
    3. Have Visual Studio Code installed.

    Creating a new repository

    1. Click "New", or click the add dropdown and choose "New repository".

    2. Make sure you are the owner.

    3. Choose a unique name for your project. Add ".github.io" to the end of the repo name if you are planning on utilizing GitHub Pages open_in_new .

    4. Choose to make your files public or private. If you want to make your repo a online webpage later, the repo must be Public

    5. Click create Create repository.

    Visit GitHub's website for more detail or click the link here open_in_new .
