    • Working with remote repositories

    As described in step 3, working with remote repositories in Visual Studio Code (VSCode) allows you to collaborate with others and manage your code on platforms like GitHub.

    Sometimes, it can be a hassle to clone a repo manually. VSCode allows you to more on a repo remotely, through the extension GitHub Repositories open_in_new !

    Add and manage remote repositores directly from VSCode

    1. Download GitHub Repositories open_in_new on VSCode.

    2. With the extension installed, click the "Open a remote window" button on the status bar.

    3. Select "Open Remote Repository..."

    4.Choose the repo you want to access. Once a repo is selected, the files will propagate in your workspace. It will look as if you cloned them onto your machine. However, you'll notice that in the bottom left corner on the status bar, it will say "GitHub".

    Push to and pull from these remotes using the VS Code interface.

    Push and pulling while working on a remote repo is exactly the same as working on cloned files.

    Take a look at step 3: Basic Git Operations for more information.
